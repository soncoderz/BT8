### hướng dẫn chạy code 

### Yêu cầu hệ thống
- Node.js (v14 trở lên)
- npm 
- file .env đã thiết lập sẳn 

### Cách chạy code:

#### 1. Cài đặt Server
```
cd server
npm install
```

#### 2. Cài đặt Client
```
cd client
npm install
```

#### 3. Chạy ứng dụng

**Chạy Server:**
```
cd server
npm run dev
```
**Chạy Client:**
```
cd client
npm start
```
